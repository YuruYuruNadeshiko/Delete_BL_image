# Delete_BL_image
정유원인지 뭐인지 자꾸 사진을 보내서 만들어 봤습니다.
제가 생각한거는 ai가 채팅방의 사진을 자동으로 판단해서 블러처리를 하는 것을 상상했지만 pc 카카오톡으로 만들기에는 무리였습니다.
고로 자동화를 만들어 봤습니다.

‼️주의사항‼️
1. 카카오톡 채팅방이 다른 창에 가려있으면 안됩니다. ai가 볼 수 없어요 ㅠㅠ
2. 완벽한 프로그램이 아니기에 정유원과 관련된 사진이 아닌 그저 닯은 사진 혹은 상관없는 사진을 제거할 수도 있습니다.
3. ai가 사진이 아닌것을 인식 할 가능성이 있습니다.
4. reference 파일도 올려두었습니다. exe 파일을 저장한 폴더와 같은 곳에 저장해주세요.
5. 응용 프로그램을 실행하면 reference 파일이 자동으로 생성이 됩니다. 거르고 싶은 사진들을 이곳에 저장해두면 자동으로 삭제를 해줍니다.
   물론 저장되어 있지 않은 비슷한 사진도 제거를 해줍니다!
7. 왠만한 사진파일은 다 인식할 것이라고 생각을 합니다만 참고용으로 인식하는 파일 확장자를 적어둡니다.
   (.jpg, .jpeg, .png, .webp, .bmp, .jfif)
8. 오류가 발생할 수 있습니다. 오류가 발생하면 알려주세요
